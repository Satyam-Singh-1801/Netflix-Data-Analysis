# Netflix-Data-Analysis
Netflix Data Analysis project using Python, Pandas, NumPy, Matplotlib, and Seaborn in Jupyter Notebook. Performed data cleaning, preprocessing, and exploratory data analysis (EDA) to uncover insights about Netflix content, ratings, release trends, genres, and Movies vs TV Shows through visualizations.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Objective
Analyze Netflix dataset and discover insights using EDA.

## Key Insights
- Movies dominate Netflix content
- TV-MA is most common rating
- USA produces most content
